\# Prep IA - Semaine 1



\## Description



Ce projet a été réalisé dans le cadre de la préparation à la formation en Intelligence Artificielle.

Il couvre les bases de Python, les modules, la programmation orientée objet, Git et Jupyter Notebook.



\---



\## Prérequis



\- Python 3.11+

\- Git

\- Jupyter Notebook



\---



\## Installation



Cloner le projet :



```bash

git clone https://github.com/Salma-JADDAR/prep-ia.git

cd prep-ia

```



Créer un environnement virtuel :



```bash

python -m venv .venv

```



Activer l'environnement :



\### Windows



```bash

.venv\\Scripts\\activate

```



Installer les dépendances :



```bash

pip install -r requirements.txt

```



\---



\## Exécution



Lancer le programme principal :



```bash

python main.py

```



Lancer Jupyter :



```bash

jupyter notebook

```
